Готовый простой сайт для Vercel/Netlify/GitHub Pages.

Файлы:
- index.html — основной сайт
- style.css — стили

Как запустить на Vercel:
1. Распакуй архив.
2. Загрузи папку/файлы в GitHub репозиторий.
3. В Vercel выбери Import Project.
4. Framework Preset: Other.
5. Deploy.
